Какие есть форматы видео?

mp4 - Является самым популярным и работает во всех современных браузерах

webm - Современный формат который не так хорошо поддерживается но у него есть ряд преимуществ такие как:

  • - Лучше поддается сжатию без потери качества, то есть видеоролик будет создавать меньше нагрузку на страницу
  • - Есть возможность делать видео на прозрачном фоне, что порой бывает крайне полезно

Какой же формат видео использовать?

Лучшим решением будет подключить оба формата, для этого нужно:

  1. Внутрь тега ... video, добавить тег ... sourse
  2. Где в атрибуте ... srs, нужно указать путь к видеофайлу
  3. Так же добавить атрибут ... type, указать значение формата видео
  4. Затем продублировать этот тег, но в атрибуте ... srs, указать путь к новому формату видеофайла, так же в атрибуте ... type, новый формат видео
  5. В теге ... video, атрибут ... srs, более не нужен, его необходимо удалить

Таким образом изначально будет загружаться видеофайл webm, но если он не поддерживается браузером то тогда загрузиться видео с форматом mp4

Атрибуты и их значения:

- srs -

Указывает путь к воспроизводимую видеоролику

- width -

Задает ширину области для воспроизведения видеоролика

- height -

Задает высоту области для воспроизведения видеоролика

- muted -

Выключает аудиодорожку

- autoplay -

Видео начинает воспроизводиться автоматически после загрузки страницы

- controls -

Добавляет панель управления к видеоролику

- loop -

Повторяет воспроизведение видео с начала после его завершения

- poster -

Указываета адрес картинки, которая будет отображаться, пока видео не доступно или не воспроизводиться

- preload -

Этот атрибут предназначен для того, что бы контролировать загрузку медиафайла

Он может иметь одно из следующий значений:

  • ... none, указывает что видео не должно быть предварительно загружено
  • ... metadata, указывает что предварительно загружаются метаданные видео, например время видео ролика
  • ... auto, указывает что весь медиафайл может быть загружен, даже если пользователь не должен его использовать
  • Пустая строка установит значение ... auto

Если не задано значение атрибута то оно определяется браузером по-умолчанию. - «То есть каждый браузер имеет по-умолчанию значение данного атрибута»

Спецификация рекомендует использовать ... metadata
